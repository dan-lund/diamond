# Diamond 💎 Easy speaker diarisation and recognition

A full-stack app that detects who spoke when in an audio file. Includes a Speaker Bank to identify known voices using cosine similarity on voice embeddings.

## Features
* Diarization: Uses pyannote/speaker-diarization-community-1 for state-of-the-art speaker segmentation.
* Speaker Identification: Compares voice signatures against a local bank voice embeddings to replace generic labels with real names.
